# Быстрый старт

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

### Windows:
```bash
run_app.bat
```

### Linux/Mac:
```bash
chmod +x run_app.sh
./run_app.sh
```

### Или напрямую:
```bash
python app.py
```

## Использование

1. Откройте браузер по адресу `http://localhost:7860`

2. **Вкладка "Настройки":**
   - Введите имя модели (например, `Qwen/Qwen2.5-7B-Instruct`)
   - Загрузите датасет в формате JSONL
   - Настройте параметры обучения

3. **Вкладка "Оценка памяти":**
   - Нажмите "Оценить память" перед запуском
   - Проверьте, достаточно ли VRAM

4. **Вкладка "Обучение":**
   - Нажмите "Начать обучение"
   - Следите за прогрессом в логах

## Формат датасета

JSONL файл, каждая строка - JSON объект:
```json
{"instruction": "Вопрос", "input": "", "output": "Ответ"}
```

## Примеры моделей

- `Qwen/Qwen2.5-7B-Instruct`
- `meta-llama/Llama-2-7b-chat-hf`
- `mistralai/Mistral-7B-Instruct-v0.2`
- Или путь к локальной модели: `./my-model`
